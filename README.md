Lunch_Tray
Aplikasi LunchTray ini dibuat sebagai latihan pemrograman mobile untuk menampilkan proses pemesanan menu makan siang. Pengguna dapat memilih makanan mulai dari entree, side dish, sampai accompaniment, kemudian melihat ringkasan dan total pesanan.
Alur Penggunaan

Pengguna menekan tombol Start Order

1. Memilih satu menu dari kategori entree

2. Melanjutkan ke pilihan side dish

3. Memilih accompaniment

4. Melihat ringkasan pesanan (subtotal, tax, total)

5. Menekan Submit Order untuk menyelesaikan pemesanan

Alur ini dibuat menggunakan navigasi antar screen sehingga pengguna bisa kembali ke langkah sebelumnya jika ingin mengubah pilihan.
